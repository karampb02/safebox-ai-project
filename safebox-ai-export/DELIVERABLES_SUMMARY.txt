PROJECT OVERVIEW:
AI-Guard: SafeBox is an AI-powered URL safety analyzer that combines fast structural 
checks with deep LLM-driven analysis to protect users from phishing and malicious 
websites. The application is fully containerized and ready for AWS deployment.

APPLICATION FEATURES
================================================================================

✓ Two-Tier Analysis Engine
  - Tier 1 (Fast Check): Structural URL analysis for common phishing indicators
  - Tier 2 (AI Analysis): LLM-powered deep evaluation of URL safety

✓ Real-Time Streaming
  - Progressive rendering of LLM responses in the UI
  - Auto-scroll enabled for continuous content flow

✓ Responsive Design
  - Optimized for both desktop and mobile browsers
  - Modern cybersecurity-themed aesthetic

✓ Security Best Practices
  - API keys managed server-side via environment variables
  - No sensitive credentials in frontend code
  - CORS properly configured

✓ Containerization
  - Single Docker image containing frontend and backend
  - Ready for AWS deployment
